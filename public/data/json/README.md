# JSON Data Files

This folder contains JSON data files used in the BiasMapper project.

## Files

### Biasmapper Files

- `biasmapper_info.json` - BiasMapper framework information and metadata
- `biasmapper_landing.json` - Landing page content data
- `biasmapper_readme.json` - README and documentation data

### News Outlet Files

- `news_*.json` - News articles and metadata from various international and Pakistani news outlets
  - `news_aljazeera.json` - Al Jazeera news data
  - `news_ary.json` - ARY News data
  - `news_bbc.json` - BBC News data
  - `news_breitbart.json` - Breitbart News data
  - `news_cnn.json` - CNN News data
  - `news_dawn.json` - Dawn News data
  - `news_dunya.json` - Dunya News data
  - `news_express.json` - Express News data
  - `news_fox.json` - Fox News data
  - `news_geo.json` - Geo News data
  - `news_guardian.json` - The Guardian data
  - `news_hum.json` - Hum News data
  - `news_msnbc.json` - MSNBC News data
  - `news_pakistantoday.json` - Pakistan Today News data
  - `news_reuters.json` - Reuters News data
  - `news_samaa.json` - Samaa News data

## Usage

These files are currently used as reference data and may be imported into the application as needed. The news data files contain sample articles and headlines from various outlets used for bias analysis demonstrations.

## Notes

- These files were moved from the project root to this organized folder structure for better project organization
- The application currently uses hardcoded data in the analyze page, but these files can be used to load dynamic data if needed
- File size: ~560KB total
